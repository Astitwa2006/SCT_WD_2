# SCT_WD_2: Calculator Web App

**SkillCraft Technology Web Development Internship — Task 02**

A fully functional calculator featuring a stark brutalist aesthetic, keyboard support, and robust arithmetic logic.

## Features
- **Brutalist UI:** Flat geometric buttons, stark `#c8ff2e` accents, and sharp corners.
- **Keyboard Support:** Type naturally using your numpad or keyboard (Enter to calculate, Backspace to delete, Escape to clear).
- **Error Handling:** Gracefully handles divide-by-zero and invalid expressions.
- **Responsive:** Scales perfectly on desktop and mobile devices.
